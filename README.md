## 🚀 Quick Start

**Get started in 2 minutes!** → [Read QUICKSTART.md](QUICKSTART.md)

Or follow these steps:

### 1. Start Backend
```bash
cd backend
python -m uvicorn main:app --reload --port 8000
```

### 2. Start Frontend
```bash
cd frontend
python -m http.server 5500
```

### 3. Access Application
```
http://localhost:5500
```

### 4. Login
- Email: `admin@ecommerce.com`
- Password: `admin123`

## ✅ Features

### 🛒 Interactive Cart System
- Add, update quantities, and remove items from your shopping cart with real-time updates.

### 👤 User Account Management
- **Profile Section**: Users can log in/sign up, view, and edit their profile details (name, address, phone).
- **Order History**: Placeholder for displaying current delivery status and past orders.
- **Wishlist/Favorites**: Placeholder for managing favorite products.

### 💳 Smooth Checkout Process
- A dedicated checkout page with order summary, including subtotal, tax, and delivery charges.

### 📱 Responsive Design
- Optimized for seamless viewing and interaction across devices (desktop, tablet, mobile).

### 🎨 Engaging UI/UX
- **Neural Background Animation**: A subtle dynamic background inspired by neural networks.
- **AOS Library**: Scroll-triggered animations for smoother experience.
- **Product Image Slideshow**: Carousel feature on product detail pages.
- **Add-to-Cart Animation**: Flying animation confirming item addition.

### 💾 Local Storage Integration
- Persists user profiles and cart data across sessions.

## 🚀 Getting Started

### 📋 Prerequisites
- Web browser (e.g., Chrome, Firefox)
- No server setup required

### 📦 Installation

```bash
git clone <repository_url>
cd TribeOf5_Ecommerce
```

Or simply download ZIP and extract.

### 💻 Usage

Open `Home_Page.html` in your browser.

**On Windows:**
```bash
start Home_Page.html
```
**On macOS:**
```bash
open Home_Page.html
```
**On Linux:**
```bash
xdg-open Home_Page.html
```

#### 📍 Navigation

- Use header links to browse Home, Shop, About, and Contact pages.
- Click the ☰ button for the sidebar:
  - Manage profile
  - View orders
  - Check wishlist
- Browse products from `Shop_Page.html`
- View product details and add to cart from `Product_Detail.html`
- Open cart sidebar, adjust quantity, or remove items
- Click Checkout to finalize order in `Check_Out.html`
